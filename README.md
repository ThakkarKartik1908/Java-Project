# Personal Movie / Media Vault

## 📌 Project Description

Personal Movie / Media Vault is a Java Application Development project for movie lovers to manage their private movie collection. Users can add movies, rate and review them, and organize their watched content easily.

The application allows users to filter movies by genre, year, or rating. It also includes a simple recommendation engine that suggests movies based on the user's favorite genres.

This project focuses on sorting, filtering, and object-based searching.

## Features

Users can:
- Add movies
- Rate and review movies
- Filter movies by genre, year, or rating
- Get top rated movies
- Get movie recommendations based on favorite genres
